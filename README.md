# Todo App

This project is just a simple todo app.

## Technologies used

+ React.js
+ MobX
+ TypeScript
+ Styled Components

## Main Features

+ Adding tasks
+ Removing tasks
+ The ability to mark a task as done
+ The navigation bar which allows you to choose the type of the tasks
